# Weather API Desktop App

Simple desktop weather application built with **Python** and **PyQt5**.
The app fetches real-time weather data from the **OpenWeatherMap API** and displays the temperature, weather description, and an emoji representing current weather conditions.

## Features

* Search weather by city name
* Temperature displayed in Celsius
* Weather description
* Weather emoji representation
* Error handling for API and connection issues
* Simple PyQt5 graphical interface

## Technologies

* Python
* PyQt5
* Requests
* OpenWeatherMap API

## Installation

Install required dependencies:

pip install -r requirements.txt

## Run the Application

python main.py

## Usage

1. Enter a city name
2. Click **Get Weather**
3. The application will display:

   * current temperature
   * weather description
   * weather emoji
